Transform evaluated width and height obtained from ff_scale_eval_dimensions into actual target width and height for scaling. Adjustment can occur if one or both of the evaluated values are of the form '-n' or if force_original_aspect_ratio is set.

Parse and evaluate string expressions for width and height. Upon success, ff_scale_adjust_dimensions must be called with evaluated width and height to obtain actual target dimensions.

Returns 0 upon success, negative value if one of the expressions could not be parsed or if NaN was the result of their evaluation.
