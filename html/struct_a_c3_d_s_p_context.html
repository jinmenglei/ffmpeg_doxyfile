Calculate bit allocation pointers. The SNR is the difference between the masking curve and the signal. AC-3 uses this value for each frequency bin to allocate bits. The snroffset parameter is a global adjustment to the SNR for all bins.

Parameters

[in]psdsignal power for each frequency bin
[in]startstarting bin location
[in]endending bin location
[in]snr_offsetSNR adjustment
[in]floornoise floor
[in]bap_tablook-up table for bit allocation pointers
[out]bapbit allocation pointers
