◆ startcode_find_candidate

int(* VC1DSPContext::startcode_find_candidate) (const uint8_t *buf, int size)

Search buf from the start for up to size bytes. Return the index of a zero byte, or >= size if not found. Ideally, use lookahead to filter out any zero bytes that are known to not be followed by one or more further zero bytes and a one byte.
