Detailed Description

This file contains a template for the codebook_trellis_rate selector function. It needs to be provided, externally, as an already included declaration, the following functions from aacenc_quantization/util.h. They're not included explicitly here to make it possible to provide alternative implementations:

  • quantize_band_cost_bits
  • abs_pow34_v structure used in optimal codebook search
