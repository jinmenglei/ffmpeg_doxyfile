◆ seek_timestamp

int64_t RTSPState::seek_timestamp

the seek value requested when calling av_seek_frame(). This value is subsequently used as part of the "Range" parameter when emitting the RTSP PLAY command. If we are currently playing, this command is called instantly. If we are currently paused, this command is called whenever we resume playback. Either way, the value is only used once, see rtsp_read_play() and rtsp_read_seek().
