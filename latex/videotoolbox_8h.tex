Allocate and initialize a Videotoolbox context.

This function should be called from the \mbox{\hyperlink{doc_2examples_2qsvdec_8c_afe1023087c26d2d410c9cd7d491c8660}{get\+\_\+format()}} callback when the caller selects the A\+V\+\_\+\+P\+I\+X\+\_\+\+F\+M\+T\+\_\+\+V\+I\+D\+E\+T\+O\+O\+L\+B\+OX format. The caller must then create the decoder object (using the output callback provided by libavcodec) that will be used for Videotoolbox-\/accelerated decoding.

When decoding with Videotoolbox is finished, the caller must destroy the decoder object and free the Videotoolbox context using \mbox{\hyperlink{tableprint__vlc_8h_a079eab092887563f2bef9106c6120089}{av\+\_\+free()}}.
