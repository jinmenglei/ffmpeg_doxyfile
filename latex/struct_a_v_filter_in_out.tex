\doxysubsection{Detailed Description}
A linked-\/list of the inputs/outputs of the filter chain.

This is mainly useful for \mbox{\hyperlink{group__lavfi_gab4718e1716c0a0d9343d49ff10d542cb}{avfilter\+\_\+graph\+\_\+parse()}} / \mbox{\hyperlink{group__lavfi_ga6c3c39e0861653c71a23f90d1397239d}{avfilter\+\_\+graph\+\_\+parse2()}}, where it is used to communicate open (unlinked) inputs and outputs from and to the caller. This struct specifies, per each not connected pad contained in the graph, the filter context and the pad index required for establishing a link.
