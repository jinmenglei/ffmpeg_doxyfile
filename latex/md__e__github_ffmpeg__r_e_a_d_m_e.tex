F\+Fmpeg is a collection of libraries and tools to process multimedia content such as audio, video, subtitles and related metadata.\hypertarget{md__e__github_ffmpeg__r_e_a_d_m_e_autotoc_md30}{}\doxysection{Libraries}\label{md__e__github_ffmpeg__r_e_a_d_m_e_autotoc_md30}

\begin{DoxyItemize}
\item {\ttfamily libavcodec} provides implementation of a wider range of codecs.
\item {\ttfamily libavformat} implements streaming protocols, container formats and basic I/O access.
\item {\ttfamily libavutil} includes hashers, decompressors and miscellaneous utility functions.
\item {\ttfamily libavfilter} provides a mean to alter decoded Audio and Video through chain of filters.
\item {\ttfamily libavdevice} provides an abstraction to access capture and playback devices.
\item {\ttfamily libswresample} implements audio mixing and resampling routines.
\item {\ttfamily libswscale} implements color conversion and scaling routines.
\end{DoxyItemize}\hypertarget{md__e__github_ffmpeg__r_e_a_d_m_e_autotoc_md31}{}\doxysection{Tools}\label{md__e__github_ffmpeg__r_e_a_d_m_e_autotoc_md31}

\begin{DoxyItemize}
\item \href{https://ffmpeg.org/ffmpeg.html}{\texttt{ ffmpeg}} is a command line toolbox to manipulate, convert and stream multimedia content.
\item \href{https://ffmpeg.org/ffplay.html}{\texttt{ ffplay}} is a minimalistic multimedia player.
\item \href{https://ffmpeg.org/ffprobe.html}{\texttt{ ffprobe}} is a simple analysis tool to inspect multimedia content.
\item Additional small tools such as {\ttfamily aviocat}, {\ttfamily ismindex} and {\ttfamily qt-\/faststart}.
\end{DoxyItemize}\hypertarget{md__e__github_ffmpeg__r_e_a_d_m_e_autotoc_md32}{}\doxysection{Documentation}\label{md__e__github_ffmpeg__r_e_a_d_m_e_autotoc_md32}
