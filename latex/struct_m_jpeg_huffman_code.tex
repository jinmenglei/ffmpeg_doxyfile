\doxysubsection{Detailed Description}
Buffer of J\+P\+EG frame data.

Optimal Huffman table generation requires the frame data to be loaded into a buffer so that the tables can be computed. There are at most mb\+\_\+width$\ast$mb\+\_\+height$\ast$12$\ast$64 of these per frame.
