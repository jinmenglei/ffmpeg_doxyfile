Find the best pixel format to convert to given a certain source pixel format. When converting from one pixel format to another, information loss may occur. For example, when converting from R\+G\+B24 to G\+R\+AY, the color information will be lost. Similarly, other losses occur when converting from some formats to other formats. \mbox{\hyperlink{group__lavc__misc__pixfmt_ga209703f647b87bd3f35ed97eb534fd21}{avcodec\+\_\+find\+\_\+best\+\_\+pix\+\_\+fmt\+\_\+of\+\_\+2()}} searches which of the given pixel formats should be used to suffer the least amount of loss. The pixel formats from which it chooses one, are determined by the pix\+\_\+fmt\+\_\+list parameter.
