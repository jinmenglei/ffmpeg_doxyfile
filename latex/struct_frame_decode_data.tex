The callback to perform some delayed processing on the frame right before it is returned to the caller.

\begin{DoxyNote}{Note}
This code is called at some unspecified point after the frame is returned from the decoder\textquotesingle{}s decode/receive\+\_\+frame call. Therefore it cannot rely on \mbox{\hyperlink{struct_a_v_codec_context}{A\+V\+Codec\+Context}} being in any specific state, so it does not get to access \mbox{\hyperlink{struct_a_v_codec_context}{A\+V\+Codec\+Context}} directly at all. All the state it needs must be stored in the post\+\_\+process\+\_\+opaque object. 
\end{DoxyNote}
