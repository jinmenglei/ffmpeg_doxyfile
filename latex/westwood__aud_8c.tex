\doxysubsection{Detailed Description}
Westwood Studios A\+UD file demuxer by Mike Melanson (\href{mailto:melanson@pcisys.net}{\texttt{ melanson@pcisys.\+net}}) for more information on the Westwood file formats, visit\+: \href{http://www.pcisys.net/~melanson/codecs/}{\texttt{ http\+://www.\+pcisys.\+net/$\sim$melanson/codecs/}} \href{http://www.geocities.com/SiliconValley/8682/aud3.txt}{\texttt{ http\+://www.\+geocities.\+com/\+Silicon\+Valley/8682/aud3.\+txt}}

Implementation note\+: There is no definite file signature for A\+UD files. The demuxer uses a probabilistic strategy for content detection. This entails performing sanity checks on certain header values in order to qualify a file. Refer to \mbox{\hyperlink{westwood__aud_8c_acb3e720f68dbc93808ea56bef95dfda9}{wsaud\+\_\+probe()}} for the precise parameters.
