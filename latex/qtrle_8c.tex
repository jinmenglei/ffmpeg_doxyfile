\doxysubsection{Detailed Description}
QT R\+LE Video \mbox{\hyperlink{struct_decoder}{Decoder}} by Mike Melanson (\href{mailto:melanson@pcisys.net}{\texttt{ melanson@pcisys.\+net}}) For more information about the QT R\+LE format, visit\+: \href{http://www.pcisys.net/~melanson/codecs/}{\texttt{ http\+://www.\+pcisys.\+net/$\sim$melanson/codecs/}}

The QT R\+LE decoder has seven modes of operation\+: 1, 2, 4, 8, 16, 24, and 32 bits per pixel. For modes 1, 2, 4, and 8 the decoder outputs P\+A\+L8 colorspace data. 16-\/bit data yields R\+G\+B555 data. 24-\/bit data is R\+G\+B24 and 32-\/bit data is R\+G\+B32.
