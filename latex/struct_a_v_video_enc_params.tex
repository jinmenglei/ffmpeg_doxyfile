\doxysubsection{Detailed Description}
Video encoding parameters for a given frame. This struct is allocated along with an optional array of per-\/block \mbox{\hyperlink{struct_a_v_video_block_params}{A\+V\+Video\+Block\+Params}} descriptors. Must be allocated with \mbox{\hyperlink{video__enc__params_8c_a794bd594f22779b9fa6e111a318b3f65}{av\+\_\+video\+\_\+enc\+\_\+params\+\_\+alloc()}}.
