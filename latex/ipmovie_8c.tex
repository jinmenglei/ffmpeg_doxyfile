\doxysubsection{Detailed Description}
Interplay M\+VE file demuxer by Mike Melanson (\href{mailto:melanson@pcisys.net}{\texttt{ melanson@pcisys.\+net}}) For more information regarding the Interplay M\+VE file format, visit\+: \href{http://www.pcisys.net/~melanson/codecs/}{\texttt{ http\+://www.\+pcisys.\+net/$\sim$melanson/codecs/}} The aforementioned site also contains a command line utility for parsing IP M\+VE files so that you can get a good idea of the typical structure of such files. This demuxer is not the best example to use if you are trying to write your own as it uses a rather roundabout approach for splitting up and sending out the chunks.
