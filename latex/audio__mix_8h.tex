Set mixing function if the parameters match.

This compares the parameters of the mixing function to the parameters in the \mbox{\hyperlink{struct_audio_mix}{Audio\+Mix}} context. If the parameters do not match, no changes are made to the active functions. If the parameters do match and the alignment is not constrained, the function is set as the generic mixing function. If the parameters match and the alignment is constrained, the function is set as the optimized mixing function.
