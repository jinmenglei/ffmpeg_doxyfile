\doxysubsection{Detailed Description}
S\+H\+A-\/1 and S\+H\+A-\/256 (Secure Hash Algorithm) hash function implementations.

This module supports the following S\+HA hash functions\+:


\begin{DoxyItemize}
\item S\+H\+A-\/1\+: 160 bits
\item S\+H\+A-\/224\+: 224 bits, as a variant of S\+H\+A-\/2
\item S\+H\+A-\/256\+: 256 bits, as a variant of S\+H\+A-\/2
\end{DoxyItemize}

\begin{DoxySeeAlso}{See also}
For S\+H\+A-\/384, S\+H\+A-\/512, and variants thereof, see \mbox{\hyperlink{group__lavu__sha512}{S\+H\+A-\/512}}. 
\end{DoxySeeAlso}
