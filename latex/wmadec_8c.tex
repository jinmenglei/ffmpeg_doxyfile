\doxysubsection{Detailed Description}
W\+MA compatible decoder. This decoder handles Microsoft Windows Media Audio data, versions 1 \& 2. W\+MA v1 is identified by audio format 0x160 in Microsoft media files (A\+S\+F/\+A\+V\+I/\+W\+AV). W\+MA v2 is identified by audio format 0x161.

To use this decoder, a calling application must supply the extra data bytes provided with the W\+MA data. These are the extra, codec-\/specific bytes at the end of a W\+A\+V\+E\+F\+O\+R\+M\+A\+T\+EX data structure. Transmit these bytes to the decoder using the extradata\mbox{[}\+\_\+size\mbox{]} fields in \mbox{\hyperlink{struct_a_v_codec_context}{A\+V\+Codec\+Context}}. There should be 4 extra bytes for v1 data and 6 extra bytes for v2 data.
