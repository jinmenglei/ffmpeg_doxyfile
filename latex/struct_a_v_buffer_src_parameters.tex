\doxysubsection{Detailed Description}
This structure contains the parameters describing the frames that will be passed to this filter.

It should be allocated with \mbox{\hyperlink{group__lavfi__buffersrc_gaae82d4f8a69757ce01421dd3167861a5}{av\+\_\+buffersrc\+\_\+parameters\+\_\+alloc()}} and freed with \mbox{\hyperlink{tableprint__vlc_8h_a079eab092887563f2bef9106c6120089}{av\+\_\+free()}}. All the allocated fields in it remain owned by the caller.
