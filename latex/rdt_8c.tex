The A\+S\+M\+Rule\+Book contains a list of comma-\/separated strings per rule, and each rule is separated by a ;. The last one also has a ; at the end so we can use it as delimiter. Every rule occurs twice, once for when the R\+T\+SP packet header marker is set and once for if it isn\textquotesingle{}t. We only read the first because we don\textquotesingle{}t care much (that\textquotesingle{}s what the \char`\"{}odd\char`\"{} variable is for). Each rule contains a set of one or more statements, optionally preceded by a single condition. If there\textquotesingle{}s a condition, the rule starts with a \textquotesingle{}\#\textquotesingle{}. Multiple conditions are merged between brackets, so there are never multiple conditions spread out over separate statements. Generally, these conditions are bitrate limits (min/max) for multi-\/bitrate streams.
