{\footnotesize\ttfamily \#define M\+A\+X\+\_\+\+D\+W\+T\+\_\+\+L\+E\+V\+E\+LS~5}

The spec limits the number of wavelet decompositions to 4 for both level 1 (V\+C-\/2) and 128 (long-\/gop default). 5 decompositions is the maximum before $>$16-\/bit buffers are needed. Schroedinger allows this for DD 9,7 and 13,7 wavelets only, limiting the others to 4 decompositions (or 3 for the fidelity filter).

We use this instead of M\+A\+X\+\_\+\+D\+E\+C\+O\+M\+P\+O\+S\+I\+T\+I\+O\+NS to save some memory.
