\doxysubsection{Detailed Description}
This structure is used to provides the necessary configurations and data to the Direct3\+D11 F\+Fmpeg \mbox{\hyperlink{struct_h_w_accel}{H\+W\+Accel}} implementation.

The application must make it available as \mbox{\hyperlink{struct_a_v_codec_context_ab222f7d747dfdceff0a76999e09648c0}{A\+V\+Codec\+Context.\+hwaccel\+\_\+context}}.

Use \mbox{\hyperlink{group__lavc__codec__hwaccel__d3d11va_ga3e098487b226ca051ac447fea036031c}{av\+\_\+d3d11va\+\_\+alloc\+\_\+context()}} exclusively to allocate an \mbox{\hyperlink{struct_a_v_d3_d11_v_a_context}{A\+V\+D3\+D11\+V\+A\+Context}}.
