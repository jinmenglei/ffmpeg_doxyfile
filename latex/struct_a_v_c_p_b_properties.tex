\doxysubsection{Detailed Description}
This structure describes the bitrate properties of an encoded bitstream. It roughly corresponds to a subset the V\+BV parameters for M\+P\+E\+G-\/2 or H\+RD parameters for H.\+264/\+H\+E\+VC.
