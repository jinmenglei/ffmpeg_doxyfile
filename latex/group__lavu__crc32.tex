\doxysubsection{Detailed Description}
C\+RC (Cyclic Redundancy Check) hash function implementation.

This module supports numerous C\+RC polynomials, in addition to the most widely used C\+R\+C-\/32-\/\+I\+E\+EE. See \mbox{\hyperlink{group__lavu__crc32_gab7717a9f1179a5ac8516982f9f2997d2}{A\+V\+C\+R\+C\+Id}} for a list of available polynomials.
