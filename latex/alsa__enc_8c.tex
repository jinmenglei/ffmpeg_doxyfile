\doxysubsection{Detailed Description}
A\+L\+SA input and output\+: output \begin{DoxyAuthor}{Author}
Luca Abeni ( lucabe72 email it ) 

Benoit Fouet ( benoit fouet free fr )
\end{DoxyAuthor}
This avdevice encoder can play audio to an A\+L\+SA (Advanced Linux Sound Architecture) device.

The filename parameter is the name of an A\+L\+SA P\+CM device capable of capture, for example \char`\"{}default\char`\"{} or \char`\"{}plughw\+:1\char`\"{}; see the A\+L\+SA documentation for naming conventions. The empty string is equivalent to \char`\"{}default\char`\"{}.

The playback period is set to the lower value available for the device, which gives a low latency suitable for real-\/time playback.
