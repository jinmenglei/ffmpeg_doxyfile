\doxysubsection{Detailed Description}
Assorted D\+P\+CM (differential pulse code modulation) audio codecs by Mike Melanson (\href{mailto:melanson@pcisys.net}{\texttt{ melanson@pcisys.\+net}}) Xan D\+P\+CM decoder by Mario Brito (\href{mailto:mbrito@student.dei.uc.pt}{\texttt{ mbrito@student.\+dei.\+uc.\+pt}}) for more information on the specific data formats, visit\+: \href{http://www.pcisys.net/~melanson/codecs/simpleaudio.html}{\texttt{ http\+://www.\+pcisys.\+net/$\sim$melanson/codecs/simpleaudio.\+html}} S\+OL D\+P\+C\+Ms implemented by Konstantin Shishkov

Note about using the Xan D\+P\+CM decoder\+: Xan D\+P\+CM is used in A\+VI files found in the Wing Commander IV computer game. These A\+VI files contain W\+A\+V\+E\+F\+O\+R\+M\+AT headers which report the audio format as 0x01\+: raw P\+CM. Clearly incorrect. To detect Xan D\+P\+CM, you will probably have to special-\/case your A\+VI demuxer to use Xan D\+P\+CM if the file uses \textquotesingle{}Xxan\textquotesingle{} (Xan video) for its video codec. Alternately, such A\+VI files also contain the fourcc \textquotesingle{}Axan\textquotesingle{} in the \textquotesingle{}auds\textquotesingle{} chunk of the A\+VI header.
