\doxysubsection{Detailed Description}
Describe a single stream, as identified by a single m= line block in the S\+DP content. In the case of R\+DT, one \mbox{\hyperlink{struct_r_t_s_p_stream}{R\+T\+S\+P\+Stream}} can represent multiple A\+V\+Streams. In this case, each \mbox{\hyperlink{struct_a_v_stream}{A\+V\+Stream}} in this set has similar content (but different codec/bitrate).
