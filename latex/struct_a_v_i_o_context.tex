\doxysubsection{Detailed Description}
Bytestream IO \mbox{\hyperlink{struct_context}{Context}}. New fields can be added to the end with minor version bumps. Removal, reordering and changes to existing fields require a major version bump. sizeof(\+A\+V\+I\+O\+Context) must not be used outside libav$\ast$.

\begin{DoxyNote}{Note}
None of the function pointers in \mbox{\hyperlink{struct_a_v_i_o_context}{A\+V\+I\+O\+Context}} should be called directly, they should only be set by the client application when implementing custom I/O. Normally these are set to the function pointers specified in \mbox{\hyperlink{avio_8h_a853f5149136a27ffba3207d8520172a5}{avio\+\_\+alloc\+\_\+context()}} 
\end{DoxyNote}
