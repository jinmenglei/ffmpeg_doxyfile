\doxysubsection{Detailed Description}
Sierra V\+MD video decoder by Vladimir \char`\"{}\+V\+A\+G\char`\"{} Gneushev (vagsoft at mail.\+ru) for more information on the Sierra V\+MD format, visit\+: \href{http://www.pcisys.net/~melanson/codecs/}{\texttt{ http\+://www.\+pcisys.\+net/$\sim$melanson/codecs/}}

The video decoder outputs P\+A\+L8 colorspace data. The decoder expects a 0x330-\/byte V\+MD file header to be transmitted via extradata during codec initialization. Each encoded frame that is sent to this decoder is expected to be prepended with the appropriate 16-\/byte frame information record from the V\+MD file.
