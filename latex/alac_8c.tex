\doxysubsection{Detailed Description}
A\+L\+AC (Apple Lossless Audio Codec) decoder \begin{DoxyAuthor}{Author}
2005 David Hammerton 
\end{DoxyAuthor}
\begin{DoxySeeAlso}{See also}
\href{http://crazney.net/programs/itunes/alac.html}{\texttt{ http\+://crazney.\+net/programs/itunes/alac.\+html}}
\end{DoxySeeAlso}
Note\+: This decoder expects a 36-\/byte Quick\+Time atom to be passed through the extradata\mbox{[}\+\_\+size\mbox{]} fields. This atom is tacked onto the end of an \textquotesingle{}alac\textquotesingle{} stsd atom and has the following format\+:

32 bits atom size 32 bits tag (\char`\"{}alac\char`\"{}) 32 bits tag version (0) 32 bits samples per frame (used when not set explicitly in the frames) 8 bits compatible version (0) 8 bits sample size 8 bits history mult (40) 8 bits initial history (10) 8 bits rice param limit (14) 8 bits channels 16 bits max\+Run (255) 32 bits max coded frame size (0 means unknown) 32 bits average bitrate (0 means unknown) 32 bits samplerate
