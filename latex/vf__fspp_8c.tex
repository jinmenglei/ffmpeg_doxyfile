\doxysubsection{Detailed Description}
Fast Simple Post-\/processing filter This implementation is based on an algorithm described in \char`\"{}\+Aria Nosratinia Embedded Post-\/\+Processing for
\+Enhancement of Compressed Images (1999)\char`\"{} (\href{http://www.utdallas.edu/~aria/papers/vlsisp99.pdf}{\texttt{ http\+://www.\+utdallas.\+edu/$\sim$aria/papers/vlsisp99.\+pdf}}) Further, with splitting (I)D\+CT into horizontal/vertical passes, one of them can be performed once per block, not per pixel. This allows for much higher speed.

Originally written by Michael Niedermayer and Nikolaj for the M\+Player project, and ported by Arwa Arif for F\+Fmpeg.
