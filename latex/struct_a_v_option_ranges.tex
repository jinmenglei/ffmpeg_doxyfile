Array of option ranges.

Most of option types use just one component. Following describes multi-\/component option types\+:

A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+M\+A\+G\+E\+\_\+\+S\+I\+ZE\+: component index 0\+: range of pixel count (width $\ast$ height). component index 1\+: range of width. component index 2\+: range of height.

\begin{DoxyNote}{Note}
