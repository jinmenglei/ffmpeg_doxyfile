Fill the frame structure variables from bitstream by parsing the given reordering table that uses the following format\+:

Each field (16 bits) in the A\+MR \mbox{\hyperlink{struct_frame}{Frame}} is stored as\+:
\begin{DoxyItemize}
\item one byte for the number of bits in the field
\item one byte for the field index
\item then, one byte for each bit of the field (from most-\/significant to least) of the position of that bit in the A\+MR frame.
\end{DoxyItemize}
