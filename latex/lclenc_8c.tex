\doxysubsection{Detailed Description}
L\+CL (Loss\+Less Codec Library) Video Codec \mbox{\hyperlink{struct_decoder}{Decoder}} for M\+S\+ZH and Z\+L\+IB codecs Experimental encoder for Z\+L\+IB R\+G\+B24

Fourcc\+: M\+S\+ZH, Z\+L\+IB

Original Win32 dll\+: Ver2.\+23 By Kenji Oshima 2000.\+09.\+20 avimszh.\+dll, avizlib.\+dll

A description of the decoding algorithm can be found here\+: \href{http://www.pcisys.net/~melanson/codecs}{\texttt{ http\+://www.\+pcisys.\+net/$\sim$melanson/codecs}}

Supports\+: B\+G\+R24 (R\+GB 24bpp)
