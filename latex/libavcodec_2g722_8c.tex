\doxysubsection{Detailed Description}
G.\+722 A\+D\+P\+CM audio codec

This G.\+722 decoder is a bit-\/exact implementation of the I\+TU G.\+722 specification for all three specified bitrates -\/ 64000bps, 56000bps and 48000bps. It passes the I\+TU tests.

\begin{DoxyNote}{Note}
For the 56000bps and 48000bps bitrates, the lowest 1 or 2 bits respectively of each byte are ignored. 
\end{DoxyNote}
