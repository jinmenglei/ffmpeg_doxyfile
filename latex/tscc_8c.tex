\doxysubsection{Detailed Description}
Tech\+Smith Camtasia decoder

Fourcc\+: T\+S\+CC

Codec is very simple\+: it codes picture (picture difference, really) with algorithm almost identical to Windows R\+L\+E8, only without padding and with greater pixel sizes, then this coded picture is packed with Z\+Lib

Supports\+: B\+G\+R8,B\+G\+R555,B\+G\+R24 -\/ only B\+G\+R8 and B\+G\+R555 tested
