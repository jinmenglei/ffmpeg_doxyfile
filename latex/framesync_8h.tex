{\footnotesize\ttfamily enum \mbox{\hyperlink{framesync_8h_af5ba5b2bb2fa0ffc56795a2d01c472cc}{F\+F\+Frame\+Sync\+Ext\+Mode}}}

This A\+PI is intended as a helper for filters that have several video input and need to combine them somehow. If the inputs have different or variable frame rate, getting the input frames to match requires a rather complex logic and a few user-\/tunable options.

In this A\+PI, when a set of synchronized input frames is ready to be procesed is called a frame event. \mbox{\hyperlink{struct_frame}{Frame}} event can be generated in response to input frames on any or all inputs and the handling of situations where some stream extend beyond the beginning or the end of others can be configured.
