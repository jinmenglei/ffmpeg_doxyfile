\doxysubsection{Detailed Description}
This struct holds all the information that needs to be passed between the caller and libavcodec for initializing Videotoolbox decoding. Its size is not a part of the public A\+BI, it must be allocated with \mbox{\hyperlink{videotoolbox_8h_aad76884fa38a53154e84e38cd4d9cc67}{av\+\_\+videotoolbox\+\_\+alloc\+\_\+context()}} and freed with \mbox{\hyperlink{tableprint__vlc_8h_a079eab092887563f2bef9106c6120089}{av\+\_\+free()}}.
