Split the commandline into an intermediate form convenient for further processing.

The commandline is assumed to be composed of options which either belong to a group (those with O\+P\+T\+\_\+\+S\+P\+EC, O\+P\+T\+\_\+\+O\+F\+F\+S\+ET or O\+P\+T\+\_\+\+P\+E\+R\+F\+I\+LE) or are global (everything else).

A group (defined by an \mbox{\hyperlink{struct_option_group_def}{Option\+Group\+Def}} struct) is a sequence of options terminated by either a group separator option (e.\+g. -\/i) or a parameter that is not an option (doesn\textquotesingle{}t start with -\/). A group without a separator option must always be first in the supplied groups list.

All options within the same group are stored in one \mbox{\hyperlink{struct_option_group}{Option\+Group}} struct in an \mbox{\hyperlink{struct_option_group_list}{Option\+Group\+List}}, all groups with the same group definition are stored in one \mbox{\hyperlink{struct_option_group_list}{Option\+Group\+List}} in \mbox{\hyperlink{struct_option_parse_context_ac495a79844c7d896aadd2d7a3cb95993}{Option\+Parse\+Context.\+groups}}.
