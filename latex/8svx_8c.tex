\doxysubsection{Detailed Description}
8svx audio decoder \begin{DoxyAuthor}{Author}
Jaikrishnan Menon
\end{DoxyAuthor}
supports\+: fibonacci delta encoding \+: exponential encoding

For more information about the 8S\+VX format\+: \href{http://netghost.narod.ru/gff/vendspec/iff/iff.txt}{\texttt{ http\+://netghost.\+narod.\+ru/gff/vendspec/iff/iff.\+txt}} \href{http://sox.sourceforge.net/AudioFormats-11.html}{\texttt{ http\+://sox.\+sourceforge.\+net/\+Audio\+Formats-\/11.\+html}} \href{http://aminet.net/package/mus/misc/wavepak}{\texttt{ http\+://aminet.\+net/package/mus/misc/wavepak}} \href{http://amigan.1emu.net/reg/8SVX.txt}{\texttt{ http\+://amigan.\+1emu.\+net/reg/8\+S\+V\+X.\+txt}}
