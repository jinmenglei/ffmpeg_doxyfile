\doxysubsection{Detailed Description}
Format I/O context. New fields can be added to the end with minor version bumps. Removal, reordering and changes to existing fields require a major version bump. sizeof(\+A\+V\+Format\+Context) must not be used outside libav$\ast$, use \mbox{\hyperlink{group__lavf__core_gac7a91abf2f59648d995894711f070f62}{avformat\+\_\+alloc\+\_\+context()}} to create an \mbox{\hyperlink{struct_a_v_format_context}{A\+V\+Format\+Context}}.

Fields can be accessed through A\+V\+Options (av\+\_\+opt$\ast$), the name string used matches the associated command line parameter name and can be found in libavformat/options\+\_\+table.\+h. The A\+V\+Option/command line parameter names differ in some cases from the C structure field names for historic reasons or brevity.
