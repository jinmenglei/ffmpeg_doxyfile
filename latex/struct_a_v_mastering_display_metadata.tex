\doxysubsection{Detailed Description}
Mastering display metadata capable of representing the color volume of the display used to master the content (S\+M\+P\+TE 2086\+:2014).

To be used as payload of a \mbox{\hyperlink{struct_a_v_frame_side_data}{A\+V\+Frame\+Side\+Data}} or \mbox{\hyperlink{struct_a_v_packet_side_data}{A\+V\+Packet\+Side\+Data}} with the appropriate type.
