\doxysubsection{Detailed Description}
Following A\+PI allows user to probe device capabilities (supported codecs, pixel formats, sample formats, resolutions, channel counts, etc). It is build on top op \mbox{\hyperlink{struct_a_v_option}{A\+V\+Option}} A\+PI. Queried capabilities make it possible to set up converters of video or audio parameters that fit to the device.

List of capabilities that can be queried\+:
\begin{DoxyItemize}
\item Capabilities valid for both audio and video devices\+:
\begin{DoxyItemize}
\item codec\+: supported audio/video codecs. type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+NT (A\+V\+Codec\+ID value)
\end{DoxyItemize}
\item Capabilities valid for audio devices\+:
\begin{DoxyItemize}
\item sample\+\_\+format\+: supported sample formats. type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+NT (A\+V\+Sample\+Format value)
\item sample\+\_\+rate\+: supported sample rates. type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+NT
\item channels\+: supported number of channels. type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+NT
\item channel\+\_\+layout\+: supported channel layouts. type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+N\+T64
\end{DoxyItemize}
\item Capabilities valid for video devices\+:
\begin{DoxyItemize}
\item pixel\+\_\+format\+: supported pixel formats. type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+NT (A\+V\+Pixel\+Format value)
\item window\+\_\+size\+: supported window sizes (describes size of the window size presented to the user). type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+M\+A\+G\+E\+\_\+\+S\+I\+ZE
\item frame\+\_\+size\+: supported frame sizes (describes size of provided video frames). type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+I\+M\+A\+G\+E\+\_\+\+S\+I\+ZE
\item fps\+: supported fps values type\+: A\+V\+\_\+\+O\+P\+T\+\_\+\+T\+Y\+P\+E\+\_\+\+R\+A\+T\+I\+O\+N\+AL
\end{DoxyItemize}
\end{DoxyItemize}

Value of the capability may be set by user using \mbox{\hyperlink{group__opt__set__funcs_ga5fd4b92bdf4f392a2847f711676a7537}{av\+\_\+opt\+\_\+set()}} function and \mbox{\hyperlink{struct_a_v_device_capabilities_query}{A\+V\+Device\+Capabilities\+Query}} object. Following queries will limit results to the values matching already set capabilities. For example, setting a codec may impact number of formats or fps values returned during next query. Setting invalid value may limit results to zero.
