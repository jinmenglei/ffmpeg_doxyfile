\doxysubsection{Detailed Description}
S\+H\+A-\/512 (Secure Hash Algorithm) hash function implementations.

This module supports the following S\+H\+A-\/2 hash functions\+:


\begin{DoxyItemize}
\item S\+H\+A-\/512/224\+: 224 bits
\item S\+H\+A-\/512/256\+: 256 bits
\item S\+H\+A-\/384\+: 384 bits
\item S\+H\+A-\/512\+: 512 bits
\end{DoxyItemize}

\begin{DoxySeeAlso}{See also}
For S\+H\+A-\/1, S\+H\+A-\/256, and variants thereof, see \mbox{\hyperlink{group__lavu__sha}{S\+HA}}. 
\end{DoxySeeAlso}
