\doxysubsection{Detailed Description}
F\+L\+I/\+F\+LC file demuxer by Mike Melanson (\href{mailto:melanson@pcisys.net}{\texttt{ melanson@pcisys.\+net}}) for more information on the .fli/.flc file format and all of its many variations, visit\+: \href{http://www.compuphase.com/flic.htm}{\texttt{ http\+://www.\+compuphase.\+com/flic.\+htm}}

This demuxer handles standard 0x\+A\+F11-\/ and 0x\+A\+F12-\/type F\+L\+Is. It also handles special F\+L\+Is from the PC games \char`\"{}\+Magic Carpet\char`\"{} and \char`\"{}\+X-\/\+C\+O\+M\+: Terror from the Deep\char`\"{}.
