\doxysubsection{Detailed Description}
Duck True\+Motion v1 Video \mbox{\hyperlink{struct_decoder}{Decoder}} by Alex Beregszaszi and Mike Melanson (\href{mailto:melanson@pcisys.net}{\texttt{ melanson@pcisys.\+net}})

The True\+Motion v1 decoder presently only decodes 16-\/bit T\+M1 data and outputs R\+G\+B555 (or R\+G\+B565) data. 24-\/bit T\+M1 data is not supported yet.
