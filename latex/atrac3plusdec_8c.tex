\doxysubsection{Detailed Description}
Sony A\+T\+R\+A\+C3+ compatible decoder.

Container formats used to store its data\+: R\+I\+FF W\+AV (.at3) and Sony Open\+MG (.oma, .aa3).

Technical description of this codec can be found here\+: \href{http://wiki.multimedia.cx/index.php?title=ATRAC3plus}{\texttt{ http\+://wiki.\+multimedia.\+cx/index.\+php?title=\+A\+T\+R\+A\+C3plus}}

Kudos to Benjamin Larsson and Michael Karcher for their precious technical help!
