\doxysubsection{Detailed Description}
F\+I\+TS image decoder

Specification\+: \href{https://fits.gsfc.nasa.gov/fits_standard.html}{\texttt{ https\+://fits.\+gsfc.\+nasa.\+gov/fits\+\_\+standard.\+html}} Version 3.\+0

Support all 2d images alongwith, bzero, bscale and blank keywords. R\+G\+BA images are supported as N\+A\+X\+I\+S3 = 3 or 4 i.\+e. Planes in R\+G\+BA order. Also C\+T\+Y\+PE = \textquotesingle{}R\+GB \textquotesingle{} should be present. Also to interpret data, values are linearly scaled using min-\/max scaling but not R\+GB images.
