\doxysubsection{Detailed Description}
P\+SX S\+TR file demuxer by Mike Melanson (\href{mailto:melanson@pcisys.net}{\texttt{ melanson@pcisys.\+net}}) This module handles streams that have been ripped from Sony Playstation CD games. This demuxer can handle either raw S\+TR files (which are just concatenations of raw compact disc sectors) or S\+TR files with 0x2\+C-\/byte R\+I\+FF headers, followed by CD sectors.
