\doxysubsection{Detailed Description}
Common code shared across all F\+Fmpeg libraries.

\begin{DoxyNote}{Note}
libavutil is designed to be modular. In most cases, in order to use the functions provided by one component of libavutil you must explicitly include the specific header containing that feature. If you are only using media-\/related components, you could simply include libavutil/avutil.\+h, which brings in most of the \char`\"{}core\char`\"{} components. 
\end{DoxyNote}
