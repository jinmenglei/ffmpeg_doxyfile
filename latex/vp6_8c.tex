\doxysubsection{Detailed Description}
V\+P6 compatible video decoder

The V\+P6F decoder accepts an optional 1 byte extradata. It is composed of\+:
\begin{DoxyItemize}
\item upper 4 bits\+: difference between encoded width and visible width
\item lower 4 bits\+: difference between encoded height and visible height 
\end{DoxyItemize}
